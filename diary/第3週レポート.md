#### 2-2. 関数load_json_paramsの作成
```python: json.py
def load_json_params(json_path):
    
    with open(json_path, "r") as f:
        input_json = json.load(f)
    
    return input_json
```

path引数に対してのjsonを読み込んで、dictを返すだけの関数を作成しました
